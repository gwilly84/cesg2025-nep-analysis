This project uses publicly available economic data from:

  - Table 36-10-0222-01: GDP by industry, constant prices

- **FRED (Federal Reserve Economic Data)**:
  - Oil Prices (WTI)
  - CAD/USD Exchange Rate

- **Bank of Canada**:
  - Historical inflation series

Note: Raw data is **not included** in this repository. See script comments for column selection and formatting details.
